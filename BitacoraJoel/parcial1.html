HTML

El HTML permite leer y entender un texto sin necesidad de seguir un orden en específico.

Tambien se utiliza el CSS y el JavaScript como apoyo/sustituto de Html.

El CSS es la apariencia y presentación de una página web. El JavaScript está más dirigido a la funcionalidad.

Html utiliza corchetes angulares para etiquetar texto, imagenes y otro contenido para mostrarlo en el navegador.

La gran mayoría de etiquetas debe tener un inicio y un fin.

El inicio de una etiqueta es normal, es decir, lleva los dos caracteres y dentro de estos se encuentra el nombre Para el cierre de la etiqueta, debe ir una diagonal al inicio del nombre Entre el inicio y final de la etiqueta debe estar el contenido de esta.

Texto en HTML

En html, hay diversas estructuras para el texto, como el “h#” para títulos (Se sustituye el # por un número del 1 al 6). Si se empieza por un "h1", en caso de volver a usar un titulo, se recomienda usar el siguiente número con tal de mantener una jerarquía.

Los parrafos deben estar determinados por una etiqueta "p".

Para separar parrafos, se usan multiples "p"

"br" es un salto de línea ; "hr" es un espacio vacío.

Tambien hay 2 tipos de listas, las ordenadas y las desordenadas. Las desordenadas comienzan con "ul", enlistando el elemento con puntos u otros simbolos. Los elementos usan "li" entre la apertura y cierre del "ul". Con las ordenadas es igual, pero cambiando el "ul" por un "ol". Los elementos "li" no pueden sacar otros elementos que no sean "li"

"b" / "strong" es texto en negritas "i" / "em" es texto en italica "u" es texto subrayado "c" es un comentario "s" / "dal" es texto tachado "q" es texto en comillas "mark" es texto marcado "small" Texto pequeño "big" Texto grande "ins" Texto insertado "sub" Texto para abajo "sup" Texto para arriba

Imagenes en HTML

Para agregar una imagen se usa la tag "img", las imagenes sirven para darle profesionalidad al texto, descansar de este y poder entender de manera grafica lo que el texto dicta.

Al poner la tag, salen dos atributos llamados “src” y “alt”, queson donde se pone la dirección de la imagen, y en alt se pone un texto alternativo en caso de que la imagen no se genere.

Se recomienda guardar las imagenes en una carpeta anidada a la carpeta de los formatos, pues de estam manera, las imagenes cargan tambien en Visual Studio y se es más accesible.

En caso de que la imagen sea online, poner el enlace o adress de la imagen en “src”. No es recomendable debido a posibles cambios de enlace o problemas de derecho de autor.

Se pueden usar los atributos “width” y “height” para cambiar el tamaño de la imagen.

Es indispensable no usar ambos al mismo tiempo, pues solo con uno, la imagen se hace proporcional de manera automática, si se usa ambos, se corre el riesgo de deformar la imagen.

Enlaces en HTML

Los enlaces son botones de texto, imagén, etc, que te mueven de una página a otra.

Para agregar un link, se usa la tag "a", que posteriormente evoluciona a “"a href=”URL”"TEXTO"/a"”

Donde dice URL, se pone el enlace, y donde dice texto, se pone un texto que será lo visible en la página.

Los enlaces tienen estados, siendo estos los no visitados, los visitados y los activos. Se diferencian con el color de cada uno, siendo estos azul, morado y rojo respectivamente. Todos los enlaces están subrayados, pero esto, junto con sus colores, pueden ca,biar con CSS.

Los enlaces con atributo target indican donde abrir dicho enlace. _self es el predeterminado, lo abre en la misma página _black lo abre en una página distinta _parent lo abre en el marco primario _top abre el documento en el cuerpo de la ventana.

Las anclas nos ayudan a enlazar con nuestra misma página. Para esto es necesario usar el atributo “id”. Al crear el enlace, se agrega un “#id” como url, lo que permite moverte desde el enlace hasta el sitio con el id correspondiente.

Apuntes Enlaces en HTML

"a href=”#""img src=”imagen.jpg”"/a" para imagenes que sean enlaces
