Position

La propiedad “position” sirve para posicionar distintos elementos en el HTM, esto a traves de CSS. La propiedad tiene cuatro formas de declararse, siendo estas “Static”, “Relative”, “Absolute” y “Sticky”.

El “position: static;” viene de manera definida en los elementos, pues es parte del flujo normal de la página, el cual es descendente. En caso de usar un “float: left”, los elementos irán en la dirección izquierda de la página (Derecha para nosotros).

El “position: absolute;” se emplea para establecer de forma exacta la posición de un elemento dentro de un contenedor que este establecido como “static”.

Un elemento “sticky” se mantiene en el mismo lugar aun si la página se desplaza. Sirve con cosas como barras de navegación y menús, aunque es más visto en los famosos pop-ups.

Apuntes Position

El “position: static” no se ve afectado por las declaraciones “top”, “bottom”, “left” y “right”, es necesario cambiar a “position: relative;” para que se ejecuten.

Animaciones

CSS permite animar la mayoría de los elementos sin usar JavaScript. Cuando especifica estilos CSS de la regla “keyframes”, la animación cambiara del estilo actual al estilo nuevo.

“Animation-delay” Sirve para retrasar la animación. “Animation-iteration-count" Sirve para ver cuantas veces se repetirá la animación. “Animation-direction" Sirve para que la imagen se mueva a una dirección (Normal, Alternate, Reverse, Alternate Reveser). “Animation-play-state" Sirve para pausar la animación.

“Animation” Propiedad rápida para configurar todas las propiedades de la animación. “Tiempo”, “Delay”, “Interation-Count” es el orden de las declaraciones.

Apuntes Animaciones

Se debe especificar cuanto tiempo debe transcurrir una animación para completar un ciclo, “animation-duration” y “animation-name” son dos etiquetas obligatorias. El valor determinado de la animacion es 0, lo que hace que no se ejecute.

Un documento de CSS puede tener varias animaciones ahí guardadas, por eso debemos ponerles nombres. Para que estas funcionen se deben vincular la animación a un elemento.

Selectores

Atributo ID es como un nombre que identifica el elemento. El valor de este atributo no puede ser duplicado. Este nombre debe ser único en todo el documento.

Atributo Class es mas flexible y puede ser asignado a varios elementos html en el documento que comparten un diseño similar.

“first-child” referencia solo al primer hijo, no necesita palabras claves o parámetros. “last-child” solo referencia al último hijo. “only-child” solo afecta cuando hay un único hijo. “not()” nos permite declarar una excepción, los estilos de la regla creada con esta pseudoclase serán asignados a todo elemento excepto aquellos incluidos en la referencia entre paréntesis. En lugar de la palabra clave de un elemento, podemos usar cualquier otra referencia.

Selector de atributo selecciona elementos html en función de sus atributos o el valor de uno de sus atributos. Esto permite aplicar estilos a los elementos que posean otros atributos. “^=” selecciona los elementos que empiezan ciertos valores “$” selecciona atributos que terminen con ciertos valores, “*=”, selecciona elementos cuyo atributo contiene una cadena de valores en cualquier posición

Los Selectores de relación en CSS permiten seleccionar elementos basados en su relación con otros elementos dentro del document, Estos selectores son útiles para aplicar estilos a elementos que tienen un vínculo estructural con otros elementos, como ancestros, padres, hermanos, o descendientes.

Selector de hijos directos (>) selecciona los elementos que son hijos inmediatos de un elemento padre.

El selector de hermanos adyacentes (+) selecciona un elemento que es un hermano inmediato, es decir, que está justo después del otro elemento

El selector de hermanos generales (~) selecciona los elementos que son hermanos de un elemento anterior, es decir, sin importar si son inmediatos o no.

Apuntes Selectores

Fieldset es un selector de elementos.

Si se utilizan las palabras “odd” y “even” en los paréntesis del child se utilizan para seleccionar elemento con base en su posición en la secuencia de hijos, específicamente para alternar entre pares e impares. Odd actúa como impar y even como par.

Un selector de elementos en CSS es una regla que selección todos los elementos de un tipo especifico en el html. Es el selector más básico.

La pseudo clase nth-child() nos permite encontrar un hijo especifico. En un div contenedor con varios parrafos, si se busca hacer referencia a uno de estos en especifico sin querer usar un id o un class, haciendo referencia al div contenedor, usar un selector del parrafo o elemento y entre los parentesis poner el numero del elemento.
